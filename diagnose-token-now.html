🔍 Token Diagnostic Tool

This tool will help diagnose why your token isn't being found.

📋 Instructions

  1. Before Login: Click "Check All Tokens" - should show nothing
  2. Login with OTP: Complete your normal login flow
  3. After OTP: Come back here and click "Check All Tokens" again
  4. Compare: See what token keys exist

🧪 Test Token

Click "Set Test Token" to manually set a token, then go to Raise Ticket page.

If the error disappears, it confirms your OTP isn't saving the token.
